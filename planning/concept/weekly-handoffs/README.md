# Weekly Concept Handoffs

A handoff translates mature concept work into recommendations for the development-planning track.

Use [the weekly handoff template](../templates/WEEKLY-HANDOFF-TEMPLATE.md).

A handoff may:

- summarize Accepted or Validated decisions;
- identify implications for future planning;
- recommend experiments or acceptance tests;
- flag contradictions with implementation;
- classify ideas as Now, Later, Never, or Needs Evidence.

A handoff must not edit `planning/active/**`, declare work complete, or authorize scope by itself.
