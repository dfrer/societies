## Session Structure Design

### Session Lengths (Bartle + Flow Theory)

| Duration | Activities | Loop Type |
|----------|-----------|-----------|
| **15-minute** | Resource gathering, quick trades, social visits | Maintenance loop |
| **2-hour** | Project completion, economic planning, political campaigns | Progression loop |
| **Multi-session** | Megaprojects, crisis response, political careers | Epic loop |

### Return Triggers (from Behavioral Economics)

1. **Completion loops**: "Just one more task..."
2. **Anticipation**: Scheduled events, expected changes
3. **Social obligation**: Community needs, relationships
4. **Investment**: Sunk cost in ongoing projects
5. **Curiosity**: "What happened while I was gone?"

## UI/UX Patterns (from Paradox)

| Pattern | Implementation | Example |
|---------|---------------|---------|
| **Layered Heatmaps** | Environmental data visualization | Pollution, resource density |
| **Predictive Feedback** | Show effects before action | Law preview, build preview |
| **Progressive Disclosure** | Complexity reveals gradually | Basic → Advanced tooltips |
| **Nested Tooltips** | Deep information on demand | Hover → Click → Deep dive |
