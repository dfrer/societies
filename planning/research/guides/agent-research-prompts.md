## Assignment Template

When assigning to an agent, use this format:

```
AGENT RESEARCH ASSIGNMENT

Agent: [Name/ID]
Task: [Task Number and Name]
Due Date: [Date]
Priority: [High/Medium/Low]

BRIEF: [One paragraph summary of what to research]

DELIVERABLES:
- [ ] Answer all research questions
- [ ] Include screenshots/examples where applicable
- [ ] Provide critical analysis (not just description)
- [ ] Make specific recommendations for Societies
- [ ] Cite all sources

TIME BUDGET: [X hours]

RESOURCES:
- Full guide: [link to specific research guide]
- Search terms: [key terms from guide]
- Target sources: [where to look]

QUALITY CHECKLIST:
- [ ] Played game for minimum required time (if applicable)
- [ ] All questions answered
- [ ] Specific examples provided
- [ ] Critical analysis included
- [ ] Recommendations actionable
- [ ] Within word count
- [ ] Properly formatted

SUBMIT TO: [location/method]
```

## Research Coordinator Checklist

### Before Assigning
- [ ] Review full research guide
- [ ] Clarify any questions
- [ ] Set clear deadline
- [ ] Define submission format

### During Research
- [ ] Check in at 50% mark
- [ ] Answer questions
- [ ] Redirect if off-track

### After Submission
- [ ] Review against quality checklist
- [ ] Request revisions if needed
- [ ] Synthesize with other research
- [ ] Update planning documents
- [ ] Acknowledge completion

---

## Research Synthesis

After all research is complete:

1. **Compile Findings**: Gather all reports
2. **Extract Insights**: Pull key learnings
3. **Identify Patterns**: Common themes across sources
4. **Update Documents**: Revise planning docs
5. **Make Decisions**: Confirm or change approaches
6. **Create Tickets**: Action items for implementation
